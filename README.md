# Pneumonia Detection Using Deep Learning

## Overview
This project detects pneumonia from chest X-ray images using Convolutional Neural Networks (CNN).

## Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- OpenCV

## Features
- Image preprocessing
- CNN model training
- Model evaluation using accuracy, precision, and recall

## How to Run
1. Open the notebook in Google Colab.
2. Run all cells step by step.
